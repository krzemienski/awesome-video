# Contributing to this Awesome List

Thank you for your interest in contributing! We welcome submissions of high-quality resources.

## How to Contribute

### Option 1: Submit via Website (Recommended)

The easiest way to contribute is through our website:

1. Visit [https://awesome.video](https://awesome.video)
2. Click on "Submit Resource" in the navigation
3. Fill out the submission form with:
   - Resource title
   - URL
   - Description
   - Category selection
4. Submit and track your submission status

### Option 2: Submit via GitHub

1. Fork the repository: [krzemienski/awesome-video](krzemienski/awesome-video)
2. Create a new branch: `git checkout -b add-resource-name`
3. Add your resource to the appropriate section in README.md
4. Follow the format: `- [Resource Name](URL) - Brief description.`
5. Make sure your addition maintains alphabetical order
6. Create a Pull Request

## Guidelines

### Quality Standards
- Resources must be actively maintained (updated within the last 2 years)
- Must be relevant to the list topic
- Should provide significant value to the community
- No promotional or marketing content

### Formatting
- Use the format: `- [Name](URL) - Description.`
- Description should be concise (1-2 sentences)
- Description must end with a period
- Maintain alphabetical order within sections
- Ensure URLs are working and use HTTPS when available

## Approval Process

1. **Submission**: Resources are submitted via website or GitHub PR
2. **Review**: Moderators review submissions for quality and relevance
3. **Feedback**: If changes are needed, feedback will be provided
4. **Approval**: Approved resources are automatically synced to the repository
5. **Sync**: The README is updated with newly approved resources

## Automatic Synchronization

This awesome list uses automatic synchronization:
- Resources approved on the website are automatically added to the README
- The repository is updated daily with newly approved resources
- Manual PRs are still welcome and will be imported into the system
